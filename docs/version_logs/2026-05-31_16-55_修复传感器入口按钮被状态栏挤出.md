# 2026-05-31 16:55 修复传感器入口按钮被状态栏挤出

## 本次修改目标

修复手机竖屏运行时，顶部状态栏文字过长导致右侧“传感器测试”入口按钮被挤出屏幕、用户看不到入口的问题。

## 修改了哪些文件

- `app/src/main/res/layout/activity_main.xml`
- `docs/version_logs/2026-05-31_16-55_修复传感器入口按钮被状态栏挤出.md`

## 每个文件主要改了什么

- `activity_main.xml`：将顶部状态栏 `tvTopStatus` 改为占用剩余宽度、单行显示并超出省略；删除原来的弹性空白占位；固定“连接”和“传感器”按钮宽度，保证竖屏也能显示。
- `docs/version_logs/2026-05-31_16-55_修复传感器入口按钮被状态栏挤出.md`：记录本次版本说明。

## 是否新增功能

否。本次是入口显示修复。

## 是否修复 bug

是。修复“传感器测试”按钮在手机竖屏顶部不可见的问题。

## 是否还有遗留问题

顶部状态栏信息在竖屏会单行省略，完整状态信息不再换行显示。这样优先保证入口按钮可见。

## 如何运行或测试这个版本

1. 在 Android Studio 中重新运行安装 App。
2. 打开 `MavlinkTest`。
3. 查看顶部右侧是否出现绿色“传感器”按钮。
4. 点击“传感器”，确认进入认证、检测编号、六路曲线和损伤列表页面。
