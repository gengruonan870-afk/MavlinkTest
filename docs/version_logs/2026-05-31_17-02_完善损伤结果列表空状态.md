# 2026-05-31 17:02 完善损伤结果列表空状态

## 本次修改目标

修复传感器测试页切换到“损伤列表”后页面空白、用户看不到表头和结果状态的问题。

## 修改了哪些文件

- `app/src/main/java/com/example/mavlinktest/SensorDataTestActivity.kt`
- `app/src/main/res/layout/activity_sensor_data_test.xml`
- `docs/version_logs/2026-05-31_17-02_完善损伤结果列表空状态.md`

## 每个文件主要改了什么

- `SensorDataTestActivity.kt`：页面初始化时立即渲染损伤结果表头；当没有数据时显示“暂无损伤结果”的提示；获取数据成功后更新“损伤结果: 共 X 条”。
- `activity_sensor_data_test.xml`：在损伤列表页新增 `tvDamageInfo`，显示当前损伤结果状态。
- `docs/version_logs/2026-05-31_17-02_完善损伤结果列表空状态.md`：记录本次版本说明。

## 是否新增功能

是。新增损伤列表空状态提示和结果数量显示。

## 是否修复 bug

是。修复未获取数据或接口返回空列表时，损伤列表页看起来完全空白的问题。

## 是否还有遗留问题

如果点击“获取数据”后仍显示暂无损伤结果，需要确认接口 `/api/damageResult` 对当前检测编号是否确实返回了损伤记录。

## 如何运行或测试这个版本

1. 重新运行安装 App。
2. 进入主界面后点击顶部“传感器”。
3. 切换到“损伤列表”，应立即看到表头和暂无数据提示。
4. 点击“获取数据”，再切换到“损伤列表”，检查是否显示“损伤结果: 共 X 条”和对应列表。
