# 2026-05-31 17:36 新增平板传感器测试布局

## 本次修改目标

为准备发送给师兄安装到平板上测试，新增传感器数据测试页的平板适配布局。手机继续使用原竖屏布局，平板自动使用更适合大屏的布局。

## 修改了哪些文件

- `app/src/main/res/layout-sw600dp/activity_sensor_data_test.xml`
- `docs/version_logs/2026-05-31_17-36_新增平板传感器测试布局.md`

## 每个文件主要改了什么

- `layout-sw600dp/activity_sensor_data_test.xml`：新增平板专用页面布局，顶部标题、状态、检测编号、认证和获取按钮横向排布；六路曲线区域扩大；损伤列表页中结果表和接口原始返回左右分栏显示。
- `docs/version_logs/2026-05-31_17-36_新增平板传感器测试布局.md`：记录本次版本说明。

## 是否新增功能

是。新增平板屏幕自动适配布局。

## 是否修复 bug

否。本次是显示适配优化。

## 是否还有遗留问题

如果师兄的平板分辨率特别小或使用强制竖屏，仍可能使用较紧凑的显示效果；但 Android 会根据屏幕宽度自动选择当前平板布局。

## 如何运行或测试这个版本

1. 在 Android Studio 中重新生成安装包。
2. 安装到平板。
3. 打开 `MavlinkTest`，点击顶部“传感器”。
4. 检查传感器测试页顶部控制区是否横向排布。
5. 点击“获取数据”，检查六路曲线是否占据更大显示区域。
6. 切换“损伤列表”，检查列表和接口返回是否左右分栏。
